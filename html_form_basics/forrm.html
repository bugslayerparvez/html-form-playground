HTML Form Tags & Attributes

Form Tag Attributes Explained

  • action: URL where the form data is sent after submission.
  • method: HTTP method (`GET` or `POST`).
  • enctype: Encoding type (`multipart/form-data` is used for file uploads).
  • target: Specifies where to display the response (`_self`, `_blank`, etc.).
  • autocomplete: Enables/disables browser autofill (`on` or `off`).
  • novalidate: Skips built-in form validation.
